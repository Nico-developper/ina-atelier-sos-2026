# Specs — ATELIER x Sound On Sound

## Livrable 1 — Top Rectangle (1 mois)

- Format : 300×250 px
- Formats acceptés : JPG / PNG / GIF (statique ou animé) / HTML5
- Poids max : 150 KB
- 1 click-through URL
- HTTPS obligatoire
- Animation : boucle max 15s (si animé)

Exports attendus :

- `deliverables/top-rectangle/top-rectangle-300x250.png` (ou .jpg / .gif)
- ou `deliverables/top-rectangle/html5/` (index.html + assets)
- `deliverables/links.txt`

---

## Livrable 2 — Dedicated E-mail Blast

- Email HTML prêt à envoyer
- Largeur : 600 px max
- Layout : tables HTML
- CSS : inline uniquement
- Images : hébergées (URLs) + ALT
- Interdits : JS, vidéo intégrée (si vidéo : image + lien)
- CTA : bouton + lien texte de secours

Exports attendus :

- `deliverables/email/email.html`
- `deliverables/links.txt`
- `deliverables/email/images-urls.txt` (si besoin)

---

## Livrable 3 — Newsletter (3 mois)

- Format : 550×260 px
- 3 versions : Mois 1 / Mois 2 / Mois 3
- ALT text requis
- Click-through : URL (unique ou par mois selon tracking)

Exports attendus :

- `deliverables/newsletter/newsletter-m1-550x260.png`
- `deliverables/newsletter/newsletter-m2-550x260.png`
- `deliverables/newsletter/newsletter-m3-550x260.png`
- `deliverables/alt-texts.txt`
- `deliverables/links.txt`
