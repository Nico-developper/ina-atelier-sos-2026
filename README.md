# INA GRM — ATELIER x Sound On Sound (Campaign Assets)

Projet : création de 3 livrables publicitaires pour la campagne media ATELIER (INA GRM) en partenariat avec Sound On Sound.

## Livrables

1. **Display Banner — Top Rectangle (1 mois)**
    - Format : 300×250 px
    - Type : statique (PNG/JPG) et/ou animé (GIF/HTML5)
    - Poids max : 150 KB

2. **Dedicated E-mail Blast**
    - Email HTML prêt à envoyer
    - Largeur max : 600 px
    - Mise en page en **tables** + CSS **inline**

3. **Newsletter — New Issue Announcement (3 mois)**
    - Format : 550×260 px
    - 3 versions : Mois 1 / Mois 2 / Mois 3

## Structure du repo

- `docs/` : brief + specs + checklist QA
- `sources/` : copy + assets (images, branding)
- `production/` : fichiers de travail (bannières, email)
- `deliverables/` : exports finaux prêts à livrer

## Workflow

1. Cadrage (specs + copy)
2. Production (exports)
3. QA (checklist)
4. Packaging livraison (deliverables)
5. Versioning Git + push GitHub

## Liens & tracking

Les URLs de redirection (click-through) et textes ALT sont fournis dans :

- `deliverables/links.txt`
- `deliverables/alt-texts.txt`

## Notes

- Compatibilité HTTPS obligatoire (assets + liens)
- Lisibilité prioritaire (texte possiblement raccourci si nécessaire)
- DA alignée sur le site ATELIER (fond noir, esthétique premium, figures de Chladni)
